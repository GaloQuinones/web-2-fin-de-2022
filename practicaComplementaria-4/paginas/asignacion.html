Entidad asignacion

Esta entidad hace referencia a las asignaciones que se encuentran disponibles dentro de un curso cada uno cuenta con atributos que los caracterizan y un id que los diferencia. En este modelo contamos con 8 atributos principales que se presentaran a continuacion

Atributos de la entidad asignacion

  • Id
  • id serie
  • id personaje
  • papel interpreta
  • tipo papel
  • fecha incio
  • fecha fin
  • temporada

Los atributos de esta entidad permiten conocer informacion importante para el desarrollo de la base de datos, la llave primaria es un Id que nos servira para generar relaciones entre tablas o entidades, las relaciones pueden ser de uno a muchos o de muchos a muchos etc.
